📘 Developer / Dokumentasi API Reseller

Base URL: {{ request.url_root.rstrip('/') }}/api/v1

Autentikasi

Authorization: Bearer RTG_API_KEY
X-API-Secret: RTG_API_SECRET
Content-Type: application/json

Jangan menaruh API Secret di JavaScript frontend. Panggil API dari server/backend milik mitra.

1. Tes API

GET /api/v1/ping

2. Cek Saldo

GET /api/v1/balance

3. Daftar Produk

GET /api/v1/products

4. Buat Order

POST /api/v1/order
{
  "product_id": 1,
  "external_ref": "INV-MITRA-0001",
  "user_id": "123456789",
  "server_id": "1234"
}

external_ref harus unik per mitra. Pengiriman ulang dengan referensi sama tidak akan memotong saldo dua kali.

5. Cek Status

GET /api/v1/order/INV-MITRA-0001

Contoh cURL

curl -X POST "{{ request.url_root.rstrip('/') }}/api/v1/order" \
  -H "Authorization: Bearer RTG_API_KEY" \
  -H "X-API-Secret: RTG_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"product_id":1,"external_ref":"INV-MITRA-0001","user_id":"123456789","server_id":"1234"}'

Alur

Website mitra → API Raja Topup Games → saldo mitra dipotong → pesanan masuk sebagai paid / processing → sistem Auto Order/provider memproses produk.

Catatan: modul ini sudah membuat API reseller, saldo, order, status, log, dan white-label metadata. Pengiriman otomatis ke Digiflazz tetap bergantung pada integrasi provider nyata di menu Auto Order.
