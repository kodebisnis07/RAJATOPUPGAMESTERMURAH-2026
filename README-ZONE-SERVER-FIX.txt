PERBAIKAN ZONE / SERVER ID

1. Kategori DANA, OVO, GoPay, ShopeePay, LinkAja, pulsa, dan sosial media tidak pernah menampilkan Zone / Server ID.
2. Game secara default hanya meminta ID Game.
3. Zone / Server ID hanya muncul bila produk di panel admin disetel ke "ID Game + Server/Zone".
4. Validasi backend memakai aturan yang sama, sehingga field server tidak diwajibkan untuk kategori non-game walaupun data lama salah.

Pengaturan produk:
Admin > Game / Produk atau Nominal > Jenis Target
- DANA/e-wallet/pulsa: Nomor HP
- Sosial media: Username / Link
- Free Fire: ID Game
- Mobile Legends (jika perlu zone): ID Game + Server/Zone
